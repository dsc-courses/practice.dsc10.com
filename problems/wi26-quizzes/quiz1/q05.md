# BEGIN PROB

For a report to the Academic Senate, you need to analyze course enrollment patterns. Write or finish **one line of Python code** that produces each quantity described.

# BEGIN SUBPROB

**(a)** A DataFrame consisting of `"Department"` as the index, and a **single column** containing the mean `"Enrollment"` of all courses within each `"Department"`.

```py
courses.get(["Department", "Enrollment"]).___________________________
```

# BEGIN SOLUTION

**Answer:** `groupby("Department").mean()` — full one-liner:

`courses.get(["Department", "Enrollment"]).groupby("Department").mean()`

Explanation: `.get(["Department", "Enrollment"])` keeps only those columns. `.groupby("Department")` groups rows by department. `.mean()` computes the mean of numeric columns per group — here mean `"Enrollment"` — producing a DataFrame with `"Department"` as the index and one column of means.

# BEGIN SOLUTION

**Answer:** `courses.plot(kind="scatter", x="Capacity", y="Enrollment")`

`kind="scatter"` draws a scatter plot; `x` and `y` name the columns for the horizontal and vertical axes.
